# telegram-bot-testing-mcp

Let AI agents test Telegram bots and Mini Apps **as a real user** — an MCP server
that drives the official Telegram Web client in a real browser (Playwright).
Structured message reads, button clicks, file uploads, screenshots.
No bot token required on the user side, no MTProto libraries.

> Status: under active development. Full documentation lands with v0.1.0.
